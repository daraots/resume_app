Automated Transcript Reports


The following application was one of my first real programming projects at AUK, it is a Visual Basic Application which scrapes data from individual spreadsheets on the MS SharePoint (instructors input grades for students), it then makes Academic Transcript Reports for each student and emails it to them individually. This transcript generator saved 2-weeks of full-time work for two administrative assistants each semester, equivalent to 160 man-hours every 4 months.
